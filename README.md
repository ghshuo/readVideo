## 前言
阅读与电影是在学习小程序过程中写的一个demo，该项是由文章阅读和电影简介两部分组成。
## 阅读
阅读文章中能够查看详细信息，文章收藏，音乐播放，分享等功能。
    
## 电影
   电影调用的接口是 豆瓣公共电影api :  https://developers.douban.com/wiki/?title=movie_v2
   电影中实现了搜索、更多、下拉分页、电影详情等功能。
